# SSD-Implementation-in-PyTorch
Implementing a pretrained SSD model in PyTorch. The some of the libraries used in this code are taken from Github repository of Alexander (Max) deGroot (https://github.com/amdegroot/ssd.pytorch). The pretrained weights used are taken from: https://s3.amazonaws.com/amdegroot-models/ssd300_mAP_77.43_v2.pth
